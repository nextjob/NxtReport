Test Report

 

Note: In order to test a report a connection to the QM Server must be possilbe.

 

Note: You MUST save the report definition prior to running the test.

 

To test a report select Test Report from the Main Menu.

 

 

The Report Test Dialog is displayed.

 

 

Select the Report Format to generate, then click the Test Report Button.

 

 

If NxtDesigner is not currently connect to the QM Server, a Connection Settings Dialog is displayed.

 

Enter the User Name, Password and Account to connect to.

 

Click Connect.

 

Check the connection status memo for the connection status.  If successful, closing the Connection Settings Dialog will result in the report being generated.

 

 

Navigate to the directory "Path for Report File" and locate "MyTestReport.xxx".

 

Open the file with the corresponding spreadsheet program and check the results.

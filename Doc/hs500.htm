QMFILES.JSON

 

The QMFILES.JSON file is a json file defining QM Data Files and their corresponding Dictionary items.  

 

This file is created by the QMBasic program CREATE_QMFILES_JSON.  It is written to the predefined location C:\QMACCOUNTS\nxtReportDefs by CREATE_QMFILES_JSON.

 

In order for NxtDesigner to locate the file, it must be copied into NxtDesigner's executable directory .

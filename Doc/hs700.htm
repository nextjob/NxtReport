Installation

 

Download the package from https://github.com/nextjob/NxtReport.

 

Compile NxtDesigner, NxtReportGen, NxtRunCommand using the Lazarus IDE.

Place the created executables into the same directory.

Copy  the correct version (32/64) of qmclilib.dll (from OpenQM install directory) into the directory now containing the executables.

 

Copy files CREATE_QMFILES_JSON and NXTREPORT into a QM accessible directory and compile and catalog the programs.

 

Create a directory that is accessible  from both the QMserver and the NxtDesigner and  NxtReportGen programs, use this directory for report creation and generation.

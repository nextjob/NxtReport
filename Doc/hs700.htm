Installation

 

Download the package from https://github.com/nextjob/NxtReport.

 

Compile NxtDesigner, NxtReportGen, NxtRunCommand using the Lazarus IDE.

Place the created executables in the same directory.

 

Copy files CREATE_QMFILES_JSON and NXTREPORT into a QM accessable directory and compile and catalog the programs.

 

Create a directory that is accessable from both the QMserver and the NxtDesigner and  NxtReportGen programs.

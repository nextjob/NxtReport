Cell Formating

 

Setting the display format for a cell.

 

Click on the cell you wish to format.

 

Click the format tool bar botton (#).

 

Select the format Category and Format from the displayed list.

 

The following slides display the PT_STD_PRICE cell being set to a currency format, and the PT_STD_DATE cell being set to a date format.

 

Note:  For MVQuery's, the cell format set at design time is copied to the cells containg the associated data at report creation.  You only need to set the format for the "place  holder" cell.

Look Up

 

The MVLookUp function defines a QM Data Read and creates a place hold in the spreadsheet for the resulting data.

 

Select the Cell that will contain the data, then select MVLookUp from the function (F(x)) drop down list.

 

 

Select the QM File to read, enter the Record Id, and the Dictionary Name for the item to return.

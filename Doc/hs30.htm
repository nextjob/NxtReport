Report Creation

 

Reports are created with NxtDesigner.  Excuting the program brings up the empty report design window.

 

 

Selecting NEW from the File Menu displays the Page Settings Dialog.

 

 

 

When creating a new report:

 

Select the orientation and paper size for the report, this will create a spreadsheet grid that corresponds to the paper size.

 

Repeat Rows, Margin Settings, Print Options can be changed at anytime.

 

Click OK to continue report creation.

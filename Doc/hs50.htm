Query 

 

The MVQuery function defines a QM Query and creates a place hold in the spreadsheet for the resulting data.

 

Select the Cell that will contain the first value of the Query data, then select MVQuery from the function (F(x)) drop down list.

 

 

The MVQuery Dialog is displayed along with the File / Dictionary Selector Dialog.

If there is a QMFILES.JSON file found in the NxtDesigner executable directory, this dialog is populated with QM Files and their associated dictionary items.

See QMFILES.JSON

 

 

 

If the File / Dictionary Selector Dialog fields have been populated, selecting the QM File Name from the QM File List will result in the corresponding Directory Items being listed in the Directory Items List box.

 

With a File Name highlighted, clicking the File Name field of the MVQuery Dialog will fill the field with the file name.

 

Highligthing a Dictionary Item and clicking the Sort By Clause or Selection(s) fields will fill the field with the dictionary name.

 

Clicking the Add Field button will add the dictionary name to the Display Fields list.

 

 

Finally clicking the OK button adds the completed Query to the report design.

 

 

Note the additional cells are filled in with the field name (Dictionary names) that will contain the report data. 

 

Format these fields (fonts, colors, styles, etc) the way in which you want the data displayed in the generated report.

 

NOTE: for dates and currencies to display correctly, you MUST define a format for the cell. See Cell Formating.

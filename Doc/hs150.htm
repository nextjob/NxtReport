Replace

 

The MVReplace function defines a tag that will be replace with data provided  by the QM program calling for the report creation. 

 

Select the Cell that will contain the data, then select MVLookUp from the function (F(x)) drop down list.

 

There are two predefined tags:

 

RUNDATE

RUNTIME

 

At report creation the QM subroutine NxtReport will replace these tags with the current system date and time.

 

For Custom tags, the QM Program calling NxtReport will need to supply a JSON variable containg the associated Tag Name Tag Value.

ie:  {"%%MYTAG1%%":"MYTAG1Value", "%%MYTAG2%%":"MYTAG2Value"}  see QMBasic Sub NxtReport
